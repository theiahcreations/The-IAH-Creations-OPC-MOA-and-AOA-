### 1. Accuracy

All updates must exactly match the data intended for official filing with the Ministry of Corporate Affairs (MCA).

### 2. Formatting

- Maintain the LaTeX structure.
- Do not remove the universal preamble.
- Ensure all subscriber details are correctly formatted within the tabular environments.

1. **Branching:** Create a feature branch for any significant structural changes (e.g., adding objects).
2. **Review:** All changes to the MOA/AOA must be peer-reviewed for typographical errors.
3. **Commit Messages:** Use clear commit messages like `Update: Add AI research object` or `Fix: Correct subscriber address`.
